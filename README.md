# 🌿 Sistema de Unidades de Conservação

Projeto desenvolvido na disciplina Hands On Work VI — UNIVALI

## 🛠️ Tecnologias
- PHP + PDO
- MySQL
- HTML / CSS

## ⚙️ Como rodar localmente
1. Clone o repositório
2. Copie `config/db.example.php` para `config/db.php`
3. Preencha suas credenciais no `db.php`
4. Importe o banco e acesse via XAMPP/Laragon
